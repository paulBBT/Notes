title: Cycle d’adoption des nouvelles technologies
type: text/vnd.mddlywiki

L’adoption d’une nouvelle technologie suit le principe suivant :

* 2,5% : Innovateurs ;
* 13,5% : Early adopters ;
* un gouffre à franchir ;
* 34% : Early Majority ;
* 34% : Late Majority ;
* 16% Laggards.
